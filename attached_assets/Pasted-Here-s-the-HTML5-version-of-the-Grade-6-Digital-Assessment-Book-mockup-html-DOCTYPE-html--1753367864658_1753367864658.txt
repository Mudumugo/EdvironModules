        function showTab(tabId) {
            // Hide all tab contents
            const tabContents = document.querySelectorAll('.tab-content');
            tabContents.forEach(tab => {
                tab.classList.remove('active');
            });
            
            // Show the selected tab content
            document.getElementById(tabId).classList.add('active');
            
            // Update active tab button
            const tabButtons = document.querySelectorAll('.tabs button');
            tabButtons.forEach(button => {
                button.classList.remove('active');
            });
            event.currentTarget.classList.add('active');
        }

        function showTerm(termId) {
            // This would handle switching between terms in the subjects view
            // In a full implementation, we would load different data for each term
            const termButtons = document.querySelectorAll('.subject-tabs button');
            termButtons.forEach(button => {
                button.classList.remove('active');
            });
            event.currentTarget.classList.add('active');

        }

## Features of this HTML5 Version:

1. **Complete Standalone File**: Works in any modern browser without dependencies
2. **Interactive Elements**:
   - Tab navigation between different sections
   - Dropdown menus for performance levels
   - Toggle buttons for behavior assessment
   - Input fields for scores and comments

3. **Responsive Design**: Adapts to different screen sizes
4. **Sample Data**: Pre-populated with example student information
5. **JavaScript Functions**:
   - `showTab()` handles navigation between main sections
   - `showTerm()` would handle term switching in the subjects view

6. **Visual Indicators**:
   - Progress bars
   - Active/inactive state styling
   - Clean, organized layout

7. **All Original Components**:
   - Student profile
   - Subject assessments
   - Behavior report
   - Summary view

To use this:
1. Copy the entire code
2. Save it as an HTML file (e.g., `assessment-book.html`)
3. Open it in any modern web browser

The design maintains all the functionality from the original PDF while making it more interactive and user-friendly. You can easily extend this by adding more subjects or terms as needed.
